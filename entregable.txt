1. ¿Qué es lo que hace bg-dark y cuál es el color hexadecimal del fondo del navbar?

La clase bg-dark de Bootstrap aplica un fondo oscuro al elemento. Al usar esta clase, el fondo se vuelve de un tono gris muy oscuro y se combina con clases como navbar-dark para que el texto se muestre en color claro y tenga buen contraste.
El color hexadecimal que utiliza bg-dark es: #212529.

2. ¿Qué es lo que permite que en el formulario la contraseña esté oculta?

La contraseña se oculta gracias al atributo type="password" en la etiqueta <input>.
Cuando el campo tiene este tipo, el navegador reemplaza los caracteres escritos por puntos o asteriscos para proteger la información.

3. ¿Dónde deberías agregar CSS en el documento si modificas el estilo de alguna clase?

Si queremos modificar estilos directamente en index.html, podemos hacerlo de tres maneras:
Dentro de una etiqueta <style> en la sección <head>.
Mediante estilos en línea (inline) usando el atributo style="" dentro de una etiqueta HTML específica.
En un archivo CSS externo, que es la forma más recomendable, enlazándolo con el index.

4. ¿Bootstrap incorpora algún mecanismo para realizar los cambios del paso 4?

Bootstrap incluye múltiples clases utilitarias que permiten modificar el diseño sin necesidad de escribir mucho CSS personalizado.
Sin embargo, si se necesita un diseño como una tarjeta con un nivel de transparencia personalizado, puede ser necesario agregar CSS adicional para lograrlo exactamente como se desea.

5. ¿Cómo cambiarías el fondo por un video?

Para colocar un video como fondo de la página, se puede usar la etiqueta <video> con propiedades como autoplay, muted y loop, posicionándolo con CSS para que cubra toda la pantalla. Otra opción es usar un iframe si el video proviene de una plataforma externa. Sin embargo, la forma más recomendable es usar la etiqueta <video> junto con CSS para que el video se adapte correctamente al tamaño de la pantalla y funcione como fondo.
